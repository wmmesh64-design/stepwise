🔑 Connect to GitHub

A GitHub Personal Access Token is needed to sync data across all users. Each team member enters their token once — it stays in their browser only.

  1. Open: github.com/settings/tokens/new
  2. Set Note: PA Tracker
  3. Check ✅ repo (full control of private repositories)
  4. Click Generate token — copy it and paste below

Your token is saved only in your browser — never shared or uploaded anywhere except GitHub API.
